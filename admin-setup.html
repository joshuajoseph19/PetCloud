Admin Account Setup

Follow these steps to create the admin account in your database

Important: You must complete these steps for admin login to work!

1 Open phpMyAdmin

Go to: http://localhost/phpmyadmin

2 Select Your Database

Click on petcloud_db (or your database name) from the left sidebar

3 Go to SQL Tab

Click the SQL tab at the top of the page

4 Copy & Run This SQL

Copy the code below and paste it into the SQL box, then click Go

-- First, add role column if it doesn't exist
ALTER TABLE `users` ADD COLUMN `role` VARCHAR(20) DEFAULT 'client' AFTER `profile_pic`;

-- Delete existing admin if any
DELETE FROM `users` WHERE email = 'admin@gmail.com';

-- Create admin user
INSERT INTO `users` (`full_name`, `email`, `password_hash`, `role`, `created_at`)
VALUES
('System Administrator', 'admin@gmail.com', '$2y$10$92IXUNpkjO0rOQ5byMi.Ye4oKoEa3Ro9llC/.og/at2.uheWG/igi', 'admin', NOW());

-- Verify
SELECT * FROM users WHERE email = 'admin@gmail.com';

If you get an error about duplicate column, that's OK! It means the role column already exists.
