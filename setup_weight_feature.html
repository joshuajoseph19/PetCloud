The weight field (in kilograms) has been added to the PetCloud adoption center. Follow these simple steps to complete the setup.

Step 1: Run Database Migration

Click the button below to update your database with the new weight field:

Run Migration Now

This will add the weight_kg column to your database.

Step 2: What's New?

After migration, you'll have access to:

  • Weight Input: Users can enter pet weight when listing pets for adoption
  • Weight Display: Weight shows on pet cards in the browse adoptions page
  • API Support: Weight data is included in all adoption listing APIs

Step 3: Test the Feature

After running the migration:

  • Go to the Pet Rehoming Form
  • Fill out the form and enter a weight (e.g., 5.5 kg)
  • Submit the form
  • Check Browse Adoptions to see the weight displayed
